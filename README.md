# Advent of Code 2024

## Build
```sh
cargo build [--release]
```

## Run

### Run single part
```sh
./target/{debug,release}/aoc2024 [day] [part] < input.txt
```

### Run single day
```sh
./target/{debug,release}/aoc2024 [day] < input.txt
```

### Run all days

Input files will be read from `./input` directory.

```sh
./target/{debug,release}/aoc2024
```
